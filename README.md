# BKITA (Bimbingan Konseling Kita)

# Deskripsi Proyek

BKITA (Bimbingan Konseling Kita) merupakan aplikasi layanan bimbingan dan konseling berbasis web yang dikembangkan menggunakan framework Laravel. Aplikasi ini membantu Guru BK dalam mengelola data siswa, menerima pengajuan konseling, melakukan komunikasi dengan siswa melalui fitur chat, serta mendokumentasikan layanan konseling secara digital.

# Fitur Utama

* Registrasi dan Login
* Dashboard Siswa
* Dashboard Guru BK
* Pengajuan Konseling
* Chat Konseling
* Riwayat Konseling
* Data Siswa
* Laporan Konseling

# Framework dan Teknologi

* Laravel
* PHP
* MySQL
* Bootstrap
* JavaScript
* XAMPP

# Cara Menjalankan Aplikasi

1. Clone repository GitHub.

2. Masuk ke folder project.

```bash
cd C:\xampp\htdocs\webk
```

3. Install dependency Laravel.

```bash
composer install
```

4. Install dependency frontend.

``` bash
npm install

5. Import database yang tersedia pada folder:

```text
database_sql/laravel.sql

6. Jalankan aplikasi.

```bash
php -S localhost:8000 -t public

7. Jalankan Vite.

```bash
npm run dev

8. Buka aplikasi melalui browser.

```text
http://localhost:8000
